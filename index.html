MindFitness AI Backend

Mental health support platform powered by Claude AI (Anthropic).

API Endpoints

POST /api/chat - Main chatbot endpoint
POST /api/vent - Vent wall with AI analysis
POST /api/toolkit - Psychological toolkit generator
POST /api/payment - Payment and premium verification
POST /api/private-chat - Peer-to-peer chat matching
POST /api/line - LINE Bot webhook

Environment Variables

Required:

  • ANTHROPIC_API_KEY - Claude API key (starts with sk-ant-)
  • DB_HOST - Database host
  • DB_USER - Database username
  • DB_PASSWORD - Database password
  • DB_NAME - Database name (default: u786472860_mindfitness_pa)

Optional:

  • LINE_CHANNEL_ACCESS_TOKEN - LINE Bot token
  • LINE_CHANNEL_SECRET - LINE Bot secret
  • PROMPTPAY_ID - PromptPay phone number
